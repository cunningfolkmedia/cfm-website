# Cunning Folk Site Suite

This repository now includes a drafted public-facing site suite for the Cunning Folk master brand, along with supporting markdown strategy documents.

## Included files

- `index.html` — homepage
- `about.html` — house philosophy / About page
- `babalon-press.html` — nonfiction imprint page
- `folk-fiction.html` — fiction imprint page
- `folk-magazine.html` — editorial imprint page
- `assets/style.css` — shared site styling
- `assets/script.js` — mobile nav, reveal animations, and dynamic year
- `voice-guide.md` — unified voice guide
- `jacket-copy-framework.md` — jacket copy and retailer copy framework

## Strategic note

The package is built around one public-facing master brand:

**Cunning Folk**

The structure assumes:

- **Cunning Folk** = the house
- **Babalon Press** = nonfiction imprint
- **FOLK Fiction** = fiction imprint
- **FOLK Magazine** = editorial imprint

## Implementation notes

- All pages share one design system and one navigation structure.
- The tone is calibrated to emphasize standards, exclusion, and editorial trust.
- Newsletter forms are front-end placeholders and should be connected to your ESP of choice.
- Cover cards are visual placeholders and should be replaced with actual title art or catalog modules.

## Recommended next step

Port the copy into your live CMS, then build title-level product pages using the house voice and jacket framework in this package.
